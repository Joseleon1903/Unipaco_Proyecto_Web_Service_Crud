Proyecto Web Service

DESCRIPCION

Este proyecto sirve como Web service del proyecto Unipago_Servicio_Cliente

Aqui se encuentra el Servicio Web denominado Ciudadano_WC en el package com.Unipago.Service

Asegurarse de hacer un Test Web service y de tener creada la base de datos para el correcto funcionamento del proyecto

Tambien se encuentra el EJB quien se encarga de la logica y de realizar las operaciones de CRUD

En el package Entity encontramos la entidad que se ocupa de mapear la entidad o tabla situada en nuestra base de dato utilizando la JPA

Se utilizo el framework EclipseLink
